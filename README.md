# 🏢 UnityNest - Society Management System

A full-stack web application developed to simplify residential society management. UnityNest provides a centralized platform for residents, administrators, and workers to manage daily society operations efficiently through a secure and user-friendly interface.

---

## 📖 Overview

UnityNest is designed to improve communication and streamline management within residential communities. The system enables residents to submit complaints, pay maintenance fees, receive society announcements, manage visitors, and participate in community voting. Administrators can efficiently manage members, workers, notices, complaints, and society records from a centralized dashboard.

---

## ✨ Features

- 🔐 Secure User Authentication
- 👥 Role-Based Access Control (Admin, Owner, Rental, Worker)
- 🏠 Member Registration & Management
- 💰 Maintenance Payment Management
- 📢 Society Notices & Announcements
- 🛠 Complaint Registration & Tracking
- 🚶 Visitor Management
- 🗳 Society Voting System
- 📄 Document Upload & Verification
- 📊 Admin Dashboard

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/KenilVegad/unitynest-society-management.git
```

2. Move the project into the XAMPP `htdocs` folder.

3. Start **Apache** and **MySQL** using XAMPP.

4. Import the database into phpMyAdmin.

5. Update your database configuration in `config.php`.

6. Open your browser and visit:

```
http://localhost/UnityNest
```
